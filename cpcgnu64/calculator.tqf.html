This report displays in the Summary table the total number of functions and the number of fuctions that have never been executed and for which we have no data. If the instrumentation have been done with the option WCET (Worst Case Execution Time), then the table contains the list of the entry points with an evaluation of the WCET for each of them. This info can be empty (and the cell is red) if we were not able to compute the WCET. This can occur when one of the called functions in the call graph starting with this entry point has never been executed.

Then the summary is followed by 2 charts:

  • % Function Time: it gives the 5 top functions that the greatest % Function Time
  • Average Function Timeit gives the 5 top functions that the greatest Average Function Time

The Functions part of the report displays all the functions found in the application with the following information:

  • Functions: the name of the function.
  • EP: is this function an entry point or not. You can provide the list of the entry points, or, if not, they are deduced from the call graph (all the functions that are never called).
  • # Calls: the number of times this function is called. If this value is 0, there is no more information for this function in the table because it has never been executed.
  • Function Time: the total time spent in this function
  • Function + Desc. Time: the total time spent in this function or one of its descendants.
  • % Function Time: the percentage of time spent in this function compared with the total time.
  • % Function + Desc. Time: the percentage of time spent in this function or one of its descendants compared with the total time.
  • Average Function Time: the average time spent in this function.

If your instrument the application with the option Compute F max and F + D max we provide the following information:

  • Max Function Time: the maximum time spent in one call of this function excluded its descendants.
  • Max Function + Desc. Time: the maximum time spent in one call of this function included its descendants.

If your instrument the application with the option WCET we provide the following information:

  • WCET: an evaluation of the Worst Case Execution Time. This info can be empty if we were not able to compute the WCET. This can occur when one of the its descendants starting has never been executed

The Call Graph part displays all the functions as an interactive call graph from left to right or from top to bottom. If the application have been instrumented with the option WCET, a tooltip on each function (node of the graph) gives the WCET.
